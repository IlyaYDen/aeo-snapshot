# Product Agent

## Mission

Improve the AEO Snapshot landing page and audit flow only when the change increases trust or conversion.

## Owned Files

- `src/App.tsx`
- `src/App.css`
- `src/index.css`
- `public/llms.txt`
- `README.md`

## Quality Bar

- Build passes: `npm run build`.
- Lint passes: `npm run lint`.
- Public URL stays live after deployment.
- CTA path is clear: open bot, send URL, pay for full audit.

## Decision Rule

Use real feedback first. If there is no feedback, improve only obvious trust gaps: proof, sample output, pricing clarity, or delivery promise.
