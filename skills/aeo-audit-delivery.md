# AEO Audit Delivery Skill

Use this when a buyer sends a website or pays for the audit.

1. Capture the customer URL, market, and competitors.
2. Check homepage metadata, schema, robots, sitemap, and `llms.txt`.
3. Search how the brand/category appears in AI-search-adjacent web results and citation surfaces.
4. Score visibility, citation readiness, entity clarity, schema readiness, and `llms.txt` readiness.
5. Write a concise PDF report with the top 10 fixes and a 30-day implementation order.
6. Include a copy-paste `llms.txt` draft.
7. Send through the Telegram bot or agreed customer channel.

Keep the report practical: no abstract SEO essay, no unverifiable guarantees.
