# No-Spam Outreach Skill

Use this before any sales action.

Allowed:

- Public product directories.
- Relevant community posts where self-promotion is allowed.
- Replies to explicit requests for SEO, AEO, GEO, AI search, or lead-gen help.
- One-to-one partner messages when the recipient clearly sells SEO, content, or web services.

Not allowed:

- Bulk Telegram cold DMs.
- Evading flood limits with alternate accounts.
- Posting identical text repeatedly.
- Misrepresenting revenue, customers, identity, or affiliations.

Every outbound action must include a clear opt-out path or be in a channel where the user explicitly asked for solutions.
